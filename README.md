# Исследование эпициклоиды
## 📝 Вступление
Программа была создана в рамках учебной практики №1, проходившей в ФГБОУ ВО «Вятский государственный университет», на кафедре прикладной математики и информатики факультета компьютерных и физико-математических наук с 29.06.2026 по 12.07.2026.

В ходе прохождения практики было необходимо разработать математическое ПО для экспериментального исследования зависимости эпициклоиды от параметров (k = R/r) в её уравнении различного вида.

## 🚀 Что умеет программа?
### Минималистичная версия (`epicycloid_mini.py`)
Данная версия представляет из себя простую визуализацию графика кривой с вводом параметра и сохранением изображения через консоль.

### Полная версия (`epicycloid.py`)
Данная версия представляет из себя полноценную интерактивную программу с визуализацией графика, анимацией для наглядности, изменением параметра и сохранением изображения через графический интерфейс.

## 🛠️ Запуск и требования
Для работы программы требуется установленный Python и библиотека `matplotlib`.
Установка библиотеки осуществляется командой:
```bash
python -m pip install matplotlib
```
Запуск самой программы командой:
```bash
python epicycloid_mini.py
# или
python epicycloid.py
```
